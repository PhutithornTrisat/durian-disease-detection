# 🌿 Durian Leaf Disease Detection System (AI)

An AI-powered system for detecting diseases in durian leaves using image analysis. Users can upload leaf images and receive automated predictions of possible diseases.

---

## 📌 Overview

This project aims to assist in identifying durian leaf diseases using Artificial Intelligence and Computer Vision. The system allows users to upload images and receive analysis results quickly, reducing the need for manual inspection.

---

## 🚀 Features

- Upload images of durian leaves for analysis  
- AI-based disease detection (via backend model)  
- Display predicted disease and confidence score  
- Responsive and user-friendly UI  
- Real-time communication with REST API  

---

## 🧠 How It Works

1. User uploads a leaf image  
2. The image is sent to the backend API  
3. The AI model processes the image  
4. The system returns prediction results (disease + confidence)  
5. Frontend displays the results to the user
